냥냥이 명령어

❗ 현재 업데이트 되지 않은 항목들을 수정중입니다 양해부탁드립니다. ❗

  • 노래채널 명령어
  • 플레이리스트 명령어
  • 기본 명령어
  • 리스트 저장 명령어
  • 접두사 명령어
명령어 설명 사용
/노래채널 set 제목만 입력해도 재생되는 채팅방을 생성합니다.
/노래채널 delete 생성했던 노래채널을 서버데이터에서 삭제합니다.
명령어 설명 사용
/리스트만들기 플레이리스트 하나를 생성합니다.
/리스트 자신이 생성한 플레이리스트 목록을 보여줍니다
/리스트정보 플레이리스트에 등록된 노래전체를 확인합니다
/리스트노래삭제 저장된 플레이리스트에서 원하는 노래 하나를 제거합니다
/리스트삭제 자신이 저장한 플레이리스트를 삭제합니다
/불러오기 자신이 저장한 플레이리스트를 불러와 대기열에 추가시킵니다
명령어 설명 사용
/재생 노래 제목이나 링크를 검색하여 재생합니다
/스킵 현재 재생중인 노래를 스킵하고 다음 노래를 재생합니다
/정지 노래를 모두 정지하고 봇이 나갑니다
/skipto (패치중..) 원하는 대기열 번호로 건너뜁니다
/셔플 대기열에 추가된 노래 순서를 섞습니다
/반복 현재곡,대기열전체 반복 기능을 킵니다
/들어와 현재 접속중인 통화방에 노래봇이 접속합니다
/나가 노래를 모두 정지하고 봇이 나갑니다
/볼륨 현재 재생중인 노래의 볼륨을 조절합니다
/접두사변경 현재 서버에서 접두사를 변경합니다
/현재곡 현재 재생중인 노래의 정보를 표기합니다
/대기열초기화 대기열을 초기화합니다 (!제거)
명령어 설명 사용
/현재노래저장 현재 재생중인 노래를 플레이리스트에 저장합니다
/대기열전체저장 대기열 노래전체를 플레이리스트에 저장합니다
명령어 설명 사용
!대기열 대기열을 표기합니다
!제거 필터 및 대기열 전체를 제거하는 메뉴를 표기합니다
